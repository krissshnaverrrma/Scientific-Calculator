Scientifc Calculator
A high-end, modular Scientific Calculator built in Java. This suite integrates mathematical, physical, and financial tools with a professional "Casio-style" interface.

## üöÄ Features
* **Dual-Line LCD Display**: View equations and results simultaneously.
* **Advanced Science Engine**: Integrated Physics ($F=ma$), Algebra (Quadratic Solver), and Financial (Simple Interest) tools.
* **Unit & Currency Converter**: Instant transformations for units and real-time currency logic.
* **Keyboard Integration**: Full support for physical typing, including shortcuts for scientific constants like $\pi$ and $c$.
* **History Logger**: Automatically saves every calculation to `calc_history.log` for auditing.
* **Custom Code-Generated Icon**: Features a perfectly centered White $\sqrt{x}$ icon generated at runtime.
